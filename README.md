# Inventaris-Barang
Sistem Inventarisasi Barang Berbasis Web dengan CodeIgniter dan Bootstrap.
<hr>

### Menu:
- Data Departemen/Unit
- Data Supplier
- Data Lokasi
- Data Jenis Pengadaan
- Data Kategori Barang
- Data Barang
- Daftar Pengadaan
- Daftar Penempatan
- Daftar Mutasi
- DaftarPeminjaman
- Daftar Pengembalian
- Daftar Retur
- Daftar Disposal
- Profil Akun
- Konfigurasi Web

### Fitur:
- Upload Gambar
- Rich Text Editor
- Multi Akun
- Group Permission
- Custom Permission
- Konfigurasi Dinamis
- Pencarian Data
- Filter Data
- Aksi Bulk Data
- Expor Excel
- Expor PDF

### Dukungan:
- XAMPP 5.6.3 - 8.1.17
##### atau
- PHP 5.6 - 8.1
- MySQL atau MariaDB
- PostgreSQL (Coming Soon)
